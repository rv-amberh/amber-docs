## Quick Start

- Install dependencies 

```bash
npm install
```
- Run the app

```bash
npm run start
```

### Other scripts

- Lint and format files

```bash
npm run lint:fix
```

## Tech Stack

- React
- react-markdown
- Tailwind CSS
- Radix UI (primitives & icons)
- ESLint & Prettier
